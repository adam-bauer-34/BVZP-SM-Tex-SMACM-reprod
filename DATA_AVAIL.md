# Data Availability Statement for "An Analytical Model for the Influence of Soil Moisture on Temperature Extremes in the Midlatitudes"

By: Adam Michael Bauer

All data used in our study are freely available online or generated by codes in this package. We specify details below.

# Observational Data
Observations for CPC data are located [here](https://downloads.psl.noaa.gov/Datasets/cpc_global_temp/). CCI data live [here](https://climate.esa.int/en/projects/soil-moisture/). The user will need to post-process these into US-country level data; reaach out to the lead author for assistance on this if needed.

# Reanalysis Data
Reanalysis data was used downloaded from [the ERA5 reanalysis portal](https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=overview). We used the following variables:
- `t2m` (two meter air temperature)
- `d2m` (two meter dew point temperature)
- `t2max` (maximum two meter air temperature)
- `swvl1` (volumetric soil moisture in the uppermost layer)
- `e` (evapotranspiration)
- `ssr` (shortwave forcing)
- `slhf` (latent heat flux)
- `str` (longwave heat flux)
- `sshf` (sensible heat flux)
- `tp` (precipitation)

We post-processed hourly values into daily means. Please reach out to the corresponding author on the paper if you need assistance in downloading the data or post-processing it into daily mean values.

# Model Simulations
Comparing our soil moisture dynamics with ERA5 requires integrating our Hasselmann model with stochastic precipitation. Please run `00_model_sim.sh` to generate these data if required.
